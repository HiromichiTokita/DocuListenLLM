DocuListen UI マニュアル

各画面の構成とボタン・機能の説明です。

1. 画面全体の構成

エリア役割
① トップバー（最上段）ファイル操作・表示切替ボタン
② テキストエリア（中央大）読み込んだテキストを表示。再生中は現在読んでいる箇所が青くハイライトされる
③ 再生コントロール（テキスト下）再生・停止・速度・音量の操作
④ 設定タブ（下段）読み上げ声の設定（ルールベース／配役）
⑤ プレイリスト（右側・任意表示）複数ファイルの連続再生リスト
⑥ 詳細設定（最下段）エンジン・AIモデル・音声品質などの設定

2. トップバー

ボタン機能
テキストを貼り付けクリップボードのテキストをそのまま読み込む。コピーしたWeb小説などを即座に再生できる
ファイルTXT / EPUB / PDF / DOCX ファイルをダイアログから開く。ファイルをウィンドウにドラッグ＆ドロップしても開ける
クリアテキストエリアを空にする
ルビ→かな（チェック）ONにすると、青空文庫形式のルビ（例: 薔薇《ばら》）を読み仮名に変換してから読み上げる
ログデバッグ用のログウィンドウを表示する（開発・動作確認用）
リストプレイリストパネルの表示／非表示を切り替える

3. 再生コントロール

操作機能
シークバー（横長スライダー）再生位置を移動する。ドラッグして任意の箇所から再生できる
◀◀（先頭ボタン）テキストの先頭（チャンク0）に戻る
■（停止ボタン）再生を停止する
▶（再生ボタン）再生を開始する。再生中に押すと一時停止／再開
速度スライダー読み上げ速度を 0.5x〜3.0x で調整（タイムストレッチ方式のため音程は変わらない）
余白スライダーチャンク（文節）間の無音時間を調整
音量スライダー再生音量を調整
位置表示（例: 3123 / 4695）現在のチャンク番号 / 全チャンク数

4. ルールベースタブ

「地の文（ナレーション）」と「セリフ（会話文）」をそれぞれ別の声で読み分ける、シンプルなモードです。AIは使いません。

設定項目説明
地の文（エンジン／キャラ／スタイル）「」や『』で囲まれていない地の文（ナレーション）を読む声を設定する
セリフ（エンジン／キャラ／スタイル）「」や『』で囲まれた会話文を読む声を設定する
▶ボタンを押すと、設定した2種類の声でそのまま再生を開始します。設定が最もシンプルで、すぐ使えます。

5. 配役タブ（AI自動配役）

ローカルAI（Ollama）がテキストを分析し、登場人物を自動的に役割分類して異なる声で読み分けるモードです。

操作の流れ

ステップボタン何をするか
①📖 自動抽出テキストを解析し、登場するキャラクターを自動検出してキャラ辞書に追加する
②🤖 AI配役キャラ辞書をもとに各セリフを「主人公・悪役・語り手…」などに自動分類する（AIがバッチで処理。500セリフ約50秒）
③▶配役結果を使って再生を開始する

その他のボタン

項目説明
声を固定（チェック）ONにすると、同じテキストを再生するとき毎回同じ声順序で再生する（ランダム変動を防ぐ）
エンジン選択（▶横）VOICEVOX / IrodoriTTS など、使用する音声エンジンを選ぶ

キャラ辞書タブ

列説明
アーキタイプ（例：主人公 女、悪役 男）AIが分類するキャラクターの役割カテゴリ。15種類あり
エンジンそのアーキタイプに使う音声エンジン（VOICEVOX等）
キャラクター実際に声を当てるVOICEVOXのキャラクター名
スタイルキャラクターの話し方（ノーマル・ささやき・感情等）

6. プレイリスト

操作説明
＋ファイルをダイアログから追加する
🗑プレイリストを全消去する
ファイル名をクリックそのファイルをテキストエリアに読み込む
×（各行右端）そのファイルをリストから削除する
▶ 連続再生リストの先頭から順番に自動で読み続ける
ドラッグ＆ドロップエクスプローラーからファイルをパネルにドロップして追加できる
プレイリストの内容はアプリ終了時に自動保存され、次回起動時に復元されます。

7. 詳細設定（最下段）

設定項目説明
CPU品質VOICEVOXの音声合成品質（Low / Medium / High）。Lowが最速、Highが最高品質（重い）
OllamaモデルAI配役に使用するLLMモデル名（例: llama3.2:latest）
速度テスト現在のOllamaモデルの処理速度を計測する
LLM確認Ollamaが起動しているか・指定モデルが存在するかをポップアップで確認する
VOICEVOXポートVOICEVOXエンジンの接続ポート番号（通常 50021 のまま変更不要）
数字の読み方「そのまま」「ひとつずつ」など数字の読み上げ方法を選択
会話括弧セリフと判定する括弧の種類（「」『』など）を設定。これ以外の括弧は地の文として扱う

8. 基本的な使い方（クイックスタート）

  1. アプリを起動する
  2. 「ファイル」ボタンでテキストファイル（TXT / EPUB / PDFなど）を開く
    　または、テキストをコピーして「テキストを貼り付け」
  3. 「ルールベース」タブで地の文・セリフの声を選ぶ
  4. ▶ボタンで再生開始
  5. 速度スライダーで好みの速度に調整する
AI配役を使う場合：
事前にOllamaを起動し「LLM確認」で動作確認 → 「📖自動抽出」→「🤖AI配役」→「▶」の順に操作してください。
